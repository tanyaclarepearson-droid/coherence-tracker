# Coherence Tracker

**Track what actually makes learning work.**

---

## 🌊 Overview

Coherence Tracker is a lightweight, tap-based classroom tool designed to help educators capture what is often invisible:

- Where energy is **held** vs **pulled away**
- How specific teaching moves influence **engagement and clarity**
- When students are truly ready to learn
- What practices lead to more **coherent, agentic learning environments**

Instead of focusing only on outcomes, this tool helps track the **conditions that make learning possible**.

---

## 🎯 Purpose

This tool was created to support a simple but powerful idea:

> Repeated, high-fidelity teaching moves—combined with noticing their effects—lead to more vibrant, engaged, and effective learning environments.

Coherence Tracker helps make those patterns visible.

---

## ⚙️ How It Works

Each entry takes ~20 seconds and captures:

### 1. Energy Direction
- Is energy **held by intention** or **pulled away**?

### 2. Field State
- Thready  
- Settling  
- Coherent  
- Vibrant  

### 3. Teaching Moves (max 2)
- Student language production  
- Breathing / regulation  
- Clarity / direction  
- Focus / stamina  
- Interaction structures  

### 4. Observed Shifts
- Engagement ↑  
- Language ↑  
- Focus ↑  
- Clarity ↑  

### 5. Student Clarity Check
- Did students know what to do?

### 6. Early Warning Signals (optional)
- Clarity off  
- Energy dropping  
- Distraction rising  
- Pacing off  

### 7. Reset Moves
- Breathing  
- Proximity  
- None  

All entries are stored locally in the browser.

---

## 🌱 Why This Matters

Most tools track:
- what was taught  
- what students produced  

This tool tracks:
- **what made learning possible in the first place**

Over time, patterns emerge:
- Which practices consistently increase engagement
- What helps students access learning more quickly
- When coherence is strongest—and when it breaks

---

## 🧠 Key Concept

At the center of this tool is one core signal:

> **Where is the energy?**

Learning strengthens when energy is:
- aligned  
- sustained  
- held by intentional structure  

Learning weakens when energy is:
- fragmented  
- pulled away  
- unsupported  

This tool helps make that visible.

---

## 🚀 Getting Started

1. Open `index.html` in a browser  
2. Use the app during or after a lesson block  
3. Log 4–6 entries per day (recommended)  
4. Review patterns over time  

Optional:
- Deploy via Netlify for mobile use  
- Add to your phone home screen for app-like access
